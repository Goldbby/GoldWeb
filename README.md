# 🌐 GoldWeb – Personal Portfolio Website

## 📌 Overview

GoldWeb is a personal portfolio website designed to showcase my skills, projects, and background as a frontend developer. This project highlights my ability to build responsive and visually appealing web interfaces using core web technologies.

## 🚀 Features

* Clean and modern user interface
* Responsive design (works on mobile and desktop)
* Organized layout for easy navigation
* Personal introduction and background
* Project showcase section

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
